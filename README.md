[express'](https://www.npmjs.com/package/express) project generator 🌵.


## ⬇ Installation

```sh
npm install -g kcts-express-generator
```

## 🛠 Quick Start

A simple and easy way to generate your project with express :

```bash
kcts projectName --flag
```

Install dependencies :

```bash
npm install
```

Start your express app :

```bash
npm run dev
```

## 🚀 Quick Start

```bash
kcts projectName --flag
cd projectName
npm install
npm run dev
```

## 💣 Command Line Options

You can configure your project with some of the following command lines

    --common, default value, create an express project with commonJs
    --module, create an express project with ECMS modules
    --ts, create an express project with ECMS modules and typescript
